装修账本

隐私政策

生效日期：2026年3月1日 | 最后更新：2026年3月3日

核心承诺：装修账本是一款完全离线的应用。所有数据均存储在您的设备本地，我们不会收集、上传或存储任何您的个人数据到我们的服务器。您的隐私是我们最重要的考虑。

1. 数据收集与存储

我们收集的数据：

  • 不收集任何个人身份信息（姓名、邮箱、电话等）
  • 不收集任何使用数据（使用习惯、点击行为等）
  • 不收集任何诊断数据（崩溃报告、性能数据等）
  • 不收集任何位置信息

数据存储方式：

  • 所有装修项目、支出记录、预算信息均存储在设备本地 SQLite 数据库
  • 图片附件存储在应用沙盒目录中
  • 所有数据完全离线，无需网络连接即可使用
  • 卸载应用将导致所有数据永久删除（符合本地存储特性）

2. 系统权限使用说明

Face ID / Touch ID（生物识别）

  • 用途：应用安全锁定，保护您的账本数据
  • 处理方式：完全由 iOS 系统处理，我们仅接收验证成功/失败的结果
  • 数据存储：不存储任何生物识别数据
  • 可选：您可以随时在设置中关闭此功能

照片库访问

  • 用途：为支出记录添加图片凭证
  • 触发方式：仅在您主动点击"添加附件"时请求
  • 数据处理：选中的图片复制到应用沙盒，不上传服务器
  • 可选：您可以拒绝此权限，不影响核心功能使用

本地通知

  • 用途：预算超支提醒
  • 处理方式：完全在设备本地生成和处理

3. 第三方服务与 SDK

我们承诺：应用不包含任何第三方数据收集服务

本应用使用的第三方库：

  • GRDB.swift（SQLite 数据库）- 完全本地运行，无网络通信

本应用不包含以下服务：

  • ✗ 统计 analytics（Google Analytics、Firebase Analytics 等）
  • ✗ 崩溃报告（Firebase Crashlytics、Bugsnag 等）
  • ✗ 广告 SDK（Google AdMob、Facebook Audience Network 等）
  • ✗ 社交分享 SDK
  • ✗ 推送通知服务（APNs 除外）

4. 应用内购买

  • 购买流程由 Apple App Store 完全处理
  • 我们仅接收购买交易的状态通知（已购买/未购买）
  • 我们不获取、不存储您的支付信息（信用卡、Apple ID 等）
  • 购买记录由 Apple 管理，符合 Apple 隐私标准

5. 数据安全与备份

安全功能：

  • 应用锁：支持密码锁和生物识别保护
  • 隐私保护：应用切换到后台时自动模糊界面
  • 数据隔离：应用沙盒保护，其他应用无法访问

备份建议：

  • 定期使用应用内导出功能备份数据（CSV/PDF）
  • iCloud 备份：如果您开启了 iCloud 备份，应用数据会随设备备份
  • 注意：卸载应用将永久删除所有数据

6. 儿童隐私保护（COPPA 合规）

装修账本符合儿童在线隐私保护法（COPPA）：

  • 不故意收集 13 岁以下儿童的任何个人信息
  • 不包含面向儿童的内容或广告
  • 如果家长发现儿童提交了个人信息，请联系我们立即删除

7. 您的权利

根据适用的隐私法规，您拥有以下权利：

  • 访问权：所有数据都在您的设备上，您可以随时查看
  • 删除权：卸载应用即可删除所有数据
  • 导出权：使用应用内导出功能获取您的数据副本
  • 更正权：您可以随时编辑或删除任何记录

8. 隐私政策更新

我们可能会更新本隐私政策以反映功能变化或法律要求：

  • 重大变更将通过应用更新说明告知
  • 最新版本隐私政策始终可在 GitHub Pages 查看
  • 继续使用应用即表示您接受更新后的政策
